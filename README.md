# JSON Editor

完整的 JSON 編輯器網頁工具，參考 [jsoneditoronline.org](https://jsoneditoronline.org/) 設計。

🔗 **線上使用**: [https://paul0728.github.io/json-validator](https://paul0728.github.io/json-validator)

## ✨ 功能

### 編輯模式

- **Text Mode** - 原始文字編輯
- **Tree Mode** - 可折疊樹狀結構視圖
- **Table Mode** - 陣列資料表格顯示

### 格式化工具

- **格式化** - 美化 JSON 縮排
- **壓縮** - 移除空白字元
- **修復** - 自動修復常見錯誤

### 驗證功能

- **JSON 驗證** - 檢查語法正確性
- **序列化驗證** - 驗證轉義後的 JSON 字串
- 錯誤位置定位、修正建議

### 進階功能

- **比較** - 比較兩個 JSON 的差異
- **查詢** - JSONPath 語法查詢
- **JSON ↔ CSV** - 格式互轉

### 檔案操作

- 拖放檔案載入
- 下載 JSON 檔案
- 複製到剪貼簿

## ⌨️ 快捷鍵

| 快捷鍵 | 功能 |
|-------|------|
| `Ctrl + Enter` | 驗證 JSON |
| `Ctrl + Shift + F` | 格式化 |
| `Escape` | 關閉對話框 |

## 🚀 本地開發

直接用瀏覽器開啟 `index.html` 即可。
